# 2. Installation ‚ö°Ô∏è

## Manual installation (recommended)

Create the themes directory if it doesn‚Äôt exist:

```bash
mkdir -p ~/.config/opencode/themes
```

Copy the theme file into the directory:

```bash
cp neofusion.json ~/.config/opencode/themes/
```

---

# 3. Basic usage ‚ú®

## Option A: Temporary (inside OpenCode)

Inside OpenCode, run:

```
/theme
```

Then select:

```
neofusion
```

---

## Option B:  Set as default

Edit your OpenCode config file:

```bash
~/.config/opencode/opencode.json
```

Add:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "theme": "neofusion"
}
```

Restart OpenCode.
